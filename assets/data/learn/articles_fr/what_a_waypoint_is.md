# Ce qu'est un point de cheminement

Un **point de cheminement** est un point de reference fixe utilise en navigation aerienne.

Au lieu de voler seulement entre un point de depart et un point d'arrivee, les avions passent souvent d'un point de cheminement a l'autre le long de leur route.

---

## 📍 1. Un point de cheminement est une position dans l'espace

Un point de cheminement est un emplacement connu, defini dans les systemes de navigation.

Il n'est peut-etre pas visible depuis le hublot, mais pour l'avion c'est une veritable reference de navigation.

Les points de cheminement aident a organiser l'endroit ou l'avion doit aller ensuite.

---

## 🧭 2. Les routes sont construites a partir de nombreux points de cheminement

Une route aerienne comprend generalement une suite de points de cheminement.

Cette suite aide a guider l'avion :

- apres le depart
- pendant la croisiere
- pendant l'arrivee

👉 Ils agissent comme des reperes le long d'une route invisible dans le ciel.

---

## 📡 3. Les systemes modernes les utilisent en permanence

Les ordinateurs de navigation et les plans de vol s'appuient fortement sur les points de cheminement.

Les equipages les utilisent pour :

- surveiller la progression sur la route
- gerer les virages
- relier les procedures entre elles

Les points de cheminement font partie des bases de la navigation aerienne moderne.

---

## ✨ Ce que cela signifie

Les points de cheminement rendent les longues routes complexes plus faciles a gerer.

Ils decoupent un trajet en points de navigation precis que les avions peuvent suivre avec exactitude.

---

## 💡 Facon simple d'y penser

Les points de cheminement, c'est comme :
> des panneaux invisibles dans le ciel...  
> qui aident a guider l'avion d'une partie de la route a la suivante.

---

## 🟢 Fait rapide

Beaucoup de noms de points de cheminement sont de courts codes de cinq lettres utilises dans la planification des vols et les systemes de navigation.

---

Un point de cheminement peut etre invisible pour les passagers -  
mais c'est l'un des elements de base qui permettent a un avion de savoir ou aller.
