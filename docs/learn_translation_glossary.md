# Learn Translation Glossary

Purpose: keep aviation and weather terminology consistent across localized Learn articles.

Scope:
- Source language: English
- Target languages: Spanish (`es`), French (`fr`), German (`de`)
- Content type: short educational markdown articles for the Learn tab

Rules:
- Prefer the glossary term unless the sentence clearly requires a different grammatical form.
- Keep terminology simple and reader-friendly. These articles are for general travelers, not pilots or engineers.
- Do not invent technical detail that is not in the source.
- If a term has several valid translations, use the preferred one below for consistency.

## Translation Preferences

Spanish:
- Prefer neutral international Spanish.
- Avoid highly regional Latin American slang.
- Keep aviation terms simple unless a technical term is already common.

French:
- Prefer modern neutral French.
- Keep explanations direct and readable, not overly formal.

German:
- Prefer plain, modern German over heavy bureaucratic wording.
- Break long compounds with natural phrasing when readability improves.

## Do Not Translate

- Article IDs
- Category IDs
- File names
- Markdown syntax
- Emoji markers in headings
- `SID`, `STAR`, `GPS`, `APU`, `IATA`, `ICAO`

## If a Term Is Missing

- Choose the clearest natural translation.
- Reuse the same choice consistently in later articles.
- Add the new term to this glossary before a large batch run.
